# Roadmap

Small, safe next steps for this project:

1. Keep the main app code in `src/`.
2. Put planning notes in `docs/` before changing code.
3. Put rough experiments in `prototypes/` until they are ready for the main app.
4. Add real assets to `assets/images/`, `assets/audio/`, or `assets/fonts/` when needed.
5. Grow the `tests/` folder as the app gains more features.

## Near-term ideas

- Add a test that checks the default routine data.
- Add a short manual testing checklist for the timer controls.
- Decide whether to keep the app as plain HTML/CSS/JS or move toward React later.
